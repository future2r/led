Objekt 'led'

Das Objekt led stellt das LED-Display dar. Es bietet Eigenschaften und Funktionen, mit denen die Darstellung angepasst werden kann und Lampen geschalten werden können.

Eigenschaften

width Breite des LED-Displays
height Höhe des LED-Displays
color Farbe der Lampen
delay Pause für Schaltvorgänge
strict Strict-Modus

Funktionen

reset() Setzt das LED-Display zurück
delay(millis) Wartet eine Anzahl von Millisekunden
on() Schaltet alle Lampen an
on(x, y) Schaltet eine Lampe an
isOn(x, y)
veraltet		 Prüft, ob eine Lampe an ist
get(x, y) Prüft, ob eine Lampe an ist
set(x, y, status) Schaltet eine Lampe an oder aus
getColor(x, y) Prüft, welche Farbe eine Lampe hat
off() Schaltet alle Lampen aus
off(x, y) Schaltet eine Lampe aus
invert() Invertiert alle Lampen
invert(x, y) Invertiert eine Lampe
writeOn(x, y, text)
veraltet		 Schaltet Lampen für einen Text ein
writeOff(x, y, text)
veraltet		 Schaltet Lampen für einen Text aus
write(x, y, status, text) Schaltet Lampen für einen Text an oder aus
move(x, y, status) Verschiebt den angezeigten Inhalt.

Eigenschaften

width

Die Eigenschaft width definiert die Anzahl der Lampen in der Breite des LED-Displays. Eine Änderung dieser Eigenschaft ändert sofort die Breite des LED-Displays auf dem Bildschirm. Der aktuelle Wert der Eigenschaft kann verwendet werden, um die aktuelle Breite des LED-Displays festzustellen. Zulässig sind Werte zwischen 1 und 200. Der Standardwert beträgt 50.

// Breite ändern
led.width = 100;

// Breite verwenden
for (int i = 0; i < led.width; i++) ...

height

Die Eigenschaft height definiert die Anzahl der Lampen in der Höhe des LED-Displays. Eine Änderung dieser Eigenschaft ändert sofort die Höhe des LED-Displays auf dem Bildschirm. Der aktuelle Wert der Eigenschaft kann verwendet werden, um die aktuelle Höhe des LED-Displays festzustellen. Zulässig sind Werte zwischen 1 und 100. Der Standardwert beträgt 10.

// Höhe ändern
led.height = 100;

// Höhe verwenden
for (int i = 0; i < led.height; i++) ...

color

Die Eigenschaft color bestimmt die Farbe der Lampen. Wenn diese Eigenschaft auf eine Farbe gesetzt wird, dann verwenden alle folgenden Funktionen diese Farbe. Die Standardfarbe ist RED.

Siehe auch:
Farben

// Farbe ändern
led.color = BLUE;

delay

Diese Eigenschaft definiert die Pause in Millisekunden, die vor jedem Schaltvorgang einer Lampe gewartet wird. Durch die Änderung dieser Eigenschaft kann die Programmgeschwindigkeit gesteuert werden. Zulässig sind Werte zwischen 0 und 60.000. Der Standardwert beträgt 20.

// Maximal Geschwindigkeit
led.delay = 0;

strict

Diese Eigenschaft aktiviert oder deaktiviert den Strict-Modus. Wenn dieser Modues aktiviert ist, dann bricht das Programm beim Zugriff auf eine Lampe außerhalb des LED-Displays mit einer Fehlermeldung ab. Standardmäßig ist dieser Modus aktiviert. Wenn der Modus deaktiviert ist, passiert beim Ein- oder Ausschalten einer Lampe außerhalb des LED-Displays nichts. Bei der Abfrage des Zustandes von Lampen außerhalb des LED-Displays gelten diese als aus.

// Strict-Modus deaktivieren
led.strict = false;

// Strict-Modus aktivieren
led.strict = true;

Funktionen

reset()

Diese Funktion setzt alle Eigenschaften wieder auf ihre Standardwerte zurück. Das LED-Display bekommt seine ursprüngliche Größe und alle Lampen sind ausgeschalten.

// LED-Display zurücksetzen
led.reset();

delay(millis)

Diese Funktion wartet für die angegebenen Millisekunden. Diese Wartezeit ist unabhängig vom aktuellen Wert der Eigenschaft delay. Zulässig sind Werte zwischen 0 und 60.000.

// 2 Sekunden warten
led.delay(2000);

on()

Diese Funktion schaltet alle Lampen des LED-Displays an. Dazu wird der aktuelle Wert der Eigenschaft color verwendet.

// Alle Lampen anschalten
led.on();

on(x, y)

Diese Funktion schaltet die Lampe in Spalte x und Zeile y an. Dazu wird der aktuelle Wert der Eigenschaft color verwendet. Die erste Zeile und Spalte haben den Wert 0. Der maximale Wert ist jeweils led.width - 1 und led.height - 1.

// Die Lampe ganz links oben anschalten
led.on(0, 0);

// Die Lampe ganz rechts unten anschalten
led.on(led.width - 1, led.height - 1);

isOn(x, y)

Diese Funktion ist veraltet. Bitte benutze jetzt get(x, y)

Über diese Funktion kann festgestellt werden, ob eine Lampe eingeschaltet ist. Wenn die Lampe eingeschaltet ist, liefert die Funktion true, andernfalls false. Die erste Zeile und Spalte haben den Wert 0. Der maximale Wert ist jeweils led.width - 1 und led.height - 1.

// Wenn die Lampe eingeschalten ist...
if (led.isOn(10, 5)) ...

// Wenn die Lampe ausgeschalten ist...
if (!led.isOn(10, 5)) ...

get(x, y)

Über diese Funktion kann festgestellt werden, ob eine Lampe eingeschaltet ist. Wenn die Lampe eingeschaltet ist, liefert die Funktion ON, andernfalls OFF. Die erste Zeile und Spalte haben den Wert 0. Der maximale Wert ist jeweils led.width - 1 und led.height - 1.

// Wenn die Lampe eingeschalten ist...
if (led.get(10, 5) == ON) ...
// oder
if (led.get(10, 5)) ...

// Wenn die Lampe ausgeschalten ist...
if (led.isOn(10, 5) == OFF) ...
// oder
if (!led.get(10, 5)) ...

set(x, y, status)

Diese Funktion schaltet die Lampe in Spalte x und Zeile y an oder aus. Dazu wird der aktuelle Wert der Eigenschaft color verwendet. Die erste Zeile und Spalte haben den Wert 0. Der maximale Wert ist jeweils led.width - 1 und led.height - 1.

// Die Lampe ganz links oben anschalten
led.set(0, 0, ON);

// Die Lampe ganz rechts unten ausschalten

getColor(x, y)

Über diese Funktion kann festgestellt werden, mit welcher Farbe eine Lampe leuchtet. Wenn die Lampe eingeschalten ist, wird die Farbe zurückgeliefert (siehe Eigenschaft color), anderfalls liefert die Funktion null. Die erste Zeile und Spalte haben den Wert 0. Der maximale Wert ist jeweils led.width - 1 und led.height - 1.

// Wenn die Lampe rot ist...
if (led.getColor(10, 5) == RED) ...

// Wenn die Lampe ausgeschalten ist...
if (led.getColor(10, 5) == null) ...

off()

Diese Funktion schaltet alle Lampen des LED-Displays aus.

// Alle Lampen ausschalten
led.off();

on(x, y)

Diese Funktion schaltet die Lampe in Spalte x und Zeile y aus. Die erste Zeile und Spalte haben den Wert 0. Der maximale Wert ist jeweils led.width - 1 und led.height - 1.

// Die Lampe ganz links oben ausschalten
led.off(0, 0);

// Die Lampe ganz rechts unten ausschalten
led.off(led.width - 1, led.height - 1);

invert()

Diese Funktion invertiert alle Lampen des LED-Displays. Dazu wird der aktuelle Wert der Eigenschaft color verwendet.

// Alle Lampen invertieren
led.invert();

invert(x, y)

Diese Funktion invertiert die Lampe in Spalte x und Zeile y. Dazu wird der aktuelle Wert der Eigenschaft color verwendet. Die erste Zeile und Spalte haben den Wert 0. Der maximale Wert ist jeweils led.width - 1 und led.height - 1.

// Die Lampe ganz links oben invertieren
led.invert(0, 0);

// Die Lampe ganz rechts unten invertieren
led.invert(led.width - 1, led.height - 1);

writeOn(x, y, text)

Diese Funktion ist veraltet. Bitte benutze jetzt write(x, y, status, text)

Diese Funktion schreibt einen Text, indem die Lampen eingeschalten werden. Dazu wird der aktuelle Wert der Eigenschaft color verwendet. Jedes Zeichen hat eine Breite von 5 Lampen und eine Höhe von 8 Lampen. Als Position wird die linke obere Lampe des ersten Zeichens verwendet. Zwischen den Zeichen wird eine Lampenspalte freigelassen. Falls ein Zeichen nicht unterstützt wird, dann wird ein Leerzeichen geschrieben.

// Das Wort 'Hallo' schreiben
led.writeOn(1, 1, "Hallo");

writeOff(x, y, text)

Diese Funktion ist veraltet. Bitte benutze jetzt write(x, y, status, text)

Diese Funktion schreibt einen Text, indem die Lampen ausgeschalten werden. Jedes Zeichen hat eine Breite von 5 Lampen und eine Höhe von 8 Lampen. Als Position wird die linke obere Lampe des ersten Zeichens verwendet. Zwischen den Zeichen wird eine Lampenspalte freigelassen. Falls ein Zeichen nicht unterstützt wird, dann wird ein Leerzeichen geschrieben.

// Das Wort 'Hallo' schreiben
led.writeOff(1, 1, "Hallo");

write(x, y, status, text)

Diese Funktion schreibt einen Text, indem die Lampen auf den angegebenen Status gesetzt werden. Jedes Zeichen hat eine Breite von 5 Lampen und eine Höhe von 8 Lampen. Als Position wird die linke obere Lampe des ersten Zeichens verwendet. Zwischen den Zeichen wird eine Lampenspalte freigelassen. Falls ein Zeichen nicht unterstützt wird, dann wird ein Leerzeichen geschrieben.

led.write(1, 1, ON, "Hallo");

move(x, y, status)

Diese Funktion verschiebt den angezeigten Inhalt in die angegebene Richtung. Alle neuen Lampen werden entsprechend dem Status gesetzt. Ein positiver x-Wert verschiebt nach rechts, ein negativer x-Wert verschiebt nach links. Ein positiver y-Wert verschiebt nach unten, ein negativer y-Wert verschiebt nach oben.

// Move 2 nach rechts und 1 nach oben
// neue Lampen bleiben aus
led.move(2, -1, OFF);
