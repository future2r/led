Object 'led'

The object led represents the LED display. It provides properties and function to adjust the display and switch the lamps.

Properties

width Width of the LED display
height Height of the LED display
color Color of the lamps
delay Pause for switching operation
strict Strict mode

Functions

reset() Resets the LED display to defaults
delay(millis) Waits the specified number of milliseconds
on() Turns on all lamps
on(x, y) Turns on one lamp
isOn(x, y)
deprecated		 Checks if a lamp is turned on
get(x, y) Checks if a lamp is turned on
set(x, y, status) Turns on or off one lamp
getColor(x, y) Checks which color a lamp has
off() Turns off all lamps
off(x, y) Turns off one lamp
invert() Inverts all lamps
invert(x, y) Inverts one lamp
writeOn(x, y, text)
deprecated		 Turns on lamps for a text
writeOff(x, y, text)
deprecated		 Turns off lamps for a text
write(x, y, status, text) Turn on or off the lamps for a text
move(x, y, status) Move the display content

Properties

width

The property width defines the number of lamps for the width of the LED display. Changing this property immediately changes the width of the LED display on the screen. The current value of this property can be used to determine the current width of the LED display. Valid value are between 1 and 200. The default value is 50.

// Change the width
led.width = 100;

// Use the width
for (int i = 0; i < led.width; i++) ...

height

The property height defines the number of lamps for the height of the LED display. Changing this property immediately changes the height of the LED display on the screen. The current value of this property can be used to determine the current height of the LED display. Valid value are between 1 and 100. The default value is 10.

// Change the height
led.height = 100;

// Use the height
for (int i = 0; i < led.height; i++) ...

color

The property color defines the color of the lamps. When this property is set to a color all succeeding operations will use this color. Die default color is RED.

See also:
Colors

// Change the color
led.color = BLUE;

delay

The property defines the pause in milliseconds the program will wait before each switching operation. By changing this property the speed of the program can be controlled. Valid values are between 0 and 60,000. The default value is 20.

// Maximal speed
led.delay = 0;

strict

This property activates or deactivates the strict mode. If this mode is activated the program terminates with an error message when a lamp outside of the LED display is accessed. By default this mode is activated. If this mode is activated, nothing will happen if a lamps outside of the LED display is turned on or off. When requesting the current state of a lamp outside of the LED display they will be considered as off.

// deactivate strict mode
led.strict = false;

// activate strict mode
led.strict = true;

Functions

reset()

This function sets all properties to their default values. The LED display gets its initial size and all lamps are off.

// Reset LED display
led.reset();

delay(millis)

This function waits for the specified number of milliseconds. The waiting time is independent of the current value of the property delay. Valid values are between 0 and 60,000.

// Wait 2 seconds
led.delay(2000);

on()

This function turns on all lamps on the LED display. This operation uses the current value of the property color.

// Turn on all lamps
led.on();

on(x, y)

This function turns on the lamp at column x and row y. This operation uses the current value of the property color. The first column and row have the value 0. The maximum value is led.width - 1 and led.height - 1 respectively.

// Turn on the lamp in the upper left corner
led.on(0, 0);

// Turn on the lamp in the lower right corner
led.on(led.width - 1, led.height - 1);

isOn(x, y)

This function is deprecated. Please use get(x, y) instead.

This function can check if a lamp is on or off. If the lamp is on this function returns true, otherwise false. The first column and row have the value 0. The maximum value is led.width - 1 and led.height - 1 respectively.

// If the lamp is on...
if (led.isOn(10, 5)) ...

// If the lamp is off...
if (!led.isOn(10, 5)) ...

See also:
Lamp Status

get(x, y)

This function can check if a lamp is on or off. If the lamp is on this function returns ON, otherwise OFF. The first column and row have the value 0. The maximum value is led.width - 1 and led.height - 1 respectively.

// If the lamp is on...
if (led.get(10, 5) == ON) ...
// or
if (led.get(10, 5)) ...

// If the lamp is off...
if (led.get(10, 5) == OFF) ...
// or
if (!led.get(10, 5)) ...

See also:
Lamp Status

set(x, y, status)

This function turns on or off the lamp at column x and row y. This operation uses the current value of the property color. The first column and row have the value 0. The maximum value is led.width - 1 and led.height - 1 respectively.

// Turn on the lamp in the upper left corner
led.set(0, 0, ON);

// Turn off the lamp in the lower right corner
led.set(led.width - 1, led.height - 1, OFF);

See also:
Lamp Status

getColor(x, y)

This function checks with which color the lamp is shining. If the lamp is turned on this function returns the color (see property color), otherwise it returns null. The first column and row have the value 0. The maximum value is led.width - 1 and led.height - 1 respectively.

// If the lamp is red...
if (led.getColor(10, 5) == RED) ...

// If the lamp is off...
if (led.getColor(10, 5) == null) ...

See also:
Colors

off()

This function turns off all lamps on the LED display.

// Turn off all lamps
led.off();

on(x, y)

This function turns off the lamp at column x and row y. The first column and row have the value 0. The maximum value is led.width - 1 and led.height - 1 respectively.

// Turn off the lamp in the upper left corner
led.off(0, 0);

// Turn off the lamp in the lower right corner
led.off(led.width - 1, led.height - 1);

invert()

This function inverts all lamps of the LED display. This operation uses the current value of the property color.

// Invert all lamps
led.invert();

invert(x, y)

This function inverts the lamp at column x and row y. This operation uses the current value of the property color. The first column and row have the value 0. The maximum value is led.width - 1 and led.height - 1 respectively.

// Invert the lamp in the upper left corner
led.invert(0, 0);

// Inverts the lamp in the lower right corner
led.invert(led.width - 1, led.height - 1);

writeOn(x, y, text)

This function is deprecated. Please use write(x, y, status, text) instead.

This function writes a text by turning on lamps. This operation uses the current value of the property color. Each character has a width of 5 lamps and a height of 8 lamps. The position is the upper left lamp of the first character. Between the characters one column will be left as a space. If a character is not supported a blank will be written.

// Write the word 'Hello'
led.writeOn(1, 1, "Hello");

writeOff(x, y, text)

This function is deprecated. Please use write(x, y, status, text) instead.

This function writes a text by turning off lamps. Each character has a width of 5 lamps and a height of 8 lamps. The position is the upper left lamp of the first character. Between the characters one column will be left as a space. If a character is not supporte a blank will be written.

// Write the word 'Hello'
led.writeOn(1, 1, "Hello");

write(x, y, status, text)

This function writes a text by turning on or off the lamps. Each character has a width of 5 lamps and a height of 8 lamps. The position is the upper left lamp of the first character. Between the characters one column will be left as a space. If a character is not supporte a blank will be written.

// Write the word 'Hello' (turn on)
led.write(1, 1, ON, "Hello");

See also:
Lamp Status

move(x, y, status)

This function moves the displayed content in the specified direction. All new lamps will set according to the status. A positive x value moves to the right, a negative x value moves to the left. A positive y value moves down, a negative y value moves up.

// Move 2 right and 1 up
// leave new lamps off
led.move(2, -1, OFF);

See also:
Lamp Status
