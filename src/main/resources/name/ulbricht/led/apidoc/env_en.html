Object 'env'

The object env provides access to the environment.

Functions

checkCancelled() Checks if the user has cancelled the program.
exit() Immediately ends the program.

Functions

checkCancelled()

This function checks if the user has cancelled the program. If so, this function will end the program immediately.

// Check if the program was cancelled
env.checkCancelled();

exit()

This function immediately ends the program.

// End the program here and now
env.exit();
