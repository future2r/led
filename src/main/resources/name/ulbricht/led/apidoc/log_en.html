Object 'log'

The object log provides functions for logging.

Functions

info(text) Writes a text as a information
warning(text) Writes a text as a warning
error(text) Writes a text as an error

Functions

info(text)

This function writes a text as a information into the log.

// Write information
log.info("Hello");

warning(text)

This function writes a text as a warning into the log.

// Write warning
log.warning("Hello");

error(text)

This function writes a text as an error into the log.

// Write error
log.error("Hello");
