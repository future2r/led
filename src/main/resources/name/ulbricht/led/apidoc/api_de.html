Objekte

led Bietet Zugriff auf das LED-Display
log Bietet Funktionen für die Protokollierung
env Bietet Zugriff auf die Umgebung

Konstanten

Lampenzustände

ON Definiert den Zustand für eine eingeschaltete Lampe. Der Wert entspricht der Konstanten true.
OFF Definiert den Zustand für eine ausgeschaltete Lampe. Der Wert entspricht der Konstanten false.

Farben

RED Definiert die Farbe Rot
GREEN Definiert die Farbe Grün
BLUE Definiert die Farbe Blau
