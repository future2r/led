Objekt 'env'

Das Objekt env bietet Zugriff auf die Umgebung.

Funktionen

checkCancelled() Prüft, ob der Anwender das Program abgebrochen hat
exit() Beendet das Programm sofort

Funktionen

checkCancelled()

Prüft, ob der Anwender das Program abgebrochen hat. Wenn das der Fall, wird das Programm sofort beendet.

// Überprüfen, ob das Programm abgebrochen wurde
env.checkCancelled();

exit()

Beendet das Programm sofort.

// Program jetzt beenden
env.exit();
